# Security Policy

## Supported versions

Security fixes are provided on the latest release line.

## Reporting a vulnerability

Do not open a public issue with exploit details.

Preferred channel:

- GitHub private advisory: https://github.com/nikhilmitrax/Zeta.js/security/advisories/new

Fallback:

- Contact the maintainer directly: `@nikhilmitrax`.

When reporting, include:

- Affected version/commit
- Reproduction steps
- Impact assessment
- Suggested mitigation (if available)

## Response expectations

- Initial acknowledgement target: within 72 hours
- Status update target: within 7 days
- Fix timeline depends on severity and scope
